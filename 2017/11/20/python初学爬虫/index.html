python初学爬虫

发表于 2017-11-20 | 分类于 爬虫

推荐学习爬虫的网站：http://cuiqingcai.com/1052.html

0、需要的知识：

http协议的理解，python基础，urllib2模块的使用，正则表达式，会使用f12工具进行分析。

1、使用urllib2进行爬虫

urllib2.urlopen（url,data,headers）

url是要爬取的地址，有时并不是浏览器顶端的网址，而是用使用f12开发工具进行查看。

data是一些参数的设置，比如登录时的用户名，密码。

headers是一些请求的头部信息，这和http协议有关。

2、爬取知乎

先模拟进行登录，登录信息使用cookie进行存储，然后再进行爬取相关的网页。

3、模拟进行登录

（1）f12打开开发工具，找到‘网络’选项，然后进行登录等操作，会显示get、post过程中传递的信息

（2）模拟登录真正的请求网址为：url=’https://www.zhihu.com/login/email'，
而且要设置cookie对登录信息进行保存，以后可以直接使用。

  • 设置cookie

  • 进行登录

（3）访问相关网页，这里我们访问编辑推荐这一页，可以看到请求网址是：zhuyeurl=‘’https://www.zhihu.com/explore/recommendations‘’

content内容服务器返回的响应信息，即下图右面的‘响应’选项中的内容。一般是网页的html内容，然后使用正则表达式re模块进行解析。

（4）如获得每个问题的题目，使用正则表达式：
首先分析html，查看标题的html形式，查看器选项可以看到每个问题在一个‘zm-item’中

正则表达式写成
zhengze='<div class="zm-item".*?>.*?<a.*?>(.*?)</a>'

.*?以最短进行匹配，（.*?）就是我们需要的问题题目内容，

正则表达式的具体使用：compile，findall是re模块的2个函数
