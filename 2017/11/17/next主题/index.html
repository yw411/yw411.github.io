github+hexo博客搭建----next主题

发表于 2017-11-17 | 分类于 搭建博客

感觉每个主题都有许多可以进行修改的地方，配置文件，安装各种插件，修改源代码等等，值得去探索。

  • _config.yml主题配置文件
  • 添加头像
  • config文件:
    • avatar字段：可以是https链接或者images/avatar.jpg(或其他格式)图像
    • menu栏：分类categories，标签tags，关于about等，比如hexo new page ‘categories’，在blog/source/categories中修改index.md文档。
    • 社交栏social link：微博、twitter、知乎等的添加
    • 友情链接栏blog rolls
    • 第三方平台添加评论、搜索、留言等
    • 音乐插件：网易云音乐生成外链，复制`<iframe>` 中的内容，添加到layout/_macro/sidebar.swig中
